# 🧠 Neural Network

A collection of Jupyter Notebooks covering core deep learning architectures and techniques — from foundational ANN concepts to advanced sequence models and unsupervised learning.

---

## 📁 Repository Structure

| Notebook | Description |
|---|---|
| `Ann.ipynb` | Artificial Neural Network — fundamentals of feedforward networks, activation functions, and backpropagation |
| `CNN.ipynb` | Convolutional Neural Network — image feature extraction using conv layers, pooling, and dense layers |
| `RNN.ipynb` | Recurrent Neural Network — sequential data modeling using vanilla RNNs |
| `matercard_using_LSTM_GRU.ipynb` | Mastercard fraud/transaction analysis using LSTM and GRU architectures |
| `Autoencoder.ipynb` | Unsupervised representation learning using encoder-decoder architecture |
| `Self_Organizing_Maps.ipynb` | Unsupervised clustering and dimensionality reduction using SOMs |
| `dl_mini_project.ipynb` | End-to-end deep learning mini project applying multiple DL concepts |
| `mlbasics.ipynb` | ML/DL fundamentals and preprocessing concepts |

---

## 🛠️ Tech Stack

- **Language:** Python 3
- **Framework:** TensorFlow / Keras
- **Environment:** Jupyter Notebook
- **Libraries:** NumPy, Pandas, Matplotlib, Scikit-learn

---

## 🚀 Getting Started

### Prerequisites

```bash
pip install tensorflow numpy pandas matplotlib scikit-learn jupyter
```

### Run Locally

```bash
git clone https://github.com/Vidit-Parekh/Neural-Network.git
cd Neural-Network
jupyter notebook
```

Then open any `.ipynb` file in your browser to explore.

## 📚 Topics Covered

- Feedforward Neural Networks (ANN)
- Convolutional Neural Networks (CNN)
- Recurrent Neural Networks (RNN)
- Long Short-Term Memory (LSTM) & Gated Recurrent Units (GRU)
- Autoencoders
- Self-Organizing Maps (SOM)
- Time-series & sequential data modeling
